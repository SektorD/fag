# Гарячие клавиши
### Выделить все:
Ctrl+A
### Копировать:
Ctrl+C
### Вставить:
Ctrl+V
### Сохранить:
Ctrl+S
### Вырезать:
Ctrl+X
### Отменить:
Ctrl+Z

# Работа с git
### Скачать проект:
```bash
git clone <URL>
```
### Учесть все файлы:
```bash
git add.
```
### Сохранить коммид:
```bash
git commid -m "..."
```
### Открыть в облако:
```bash
git push
```
### Принять изменение из облока:
```bash
git pull
```
